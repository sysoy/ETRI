IBM Blockchain Platform(IBP)

Relationship between IBP and Hyperledger Fabric

IBM provides its‘ IBM blockchain platform, which is based on Hyperledger Fabric.

Hyperledger Fabric is a formal Linux Foundation project which IBM and other organisations are members of.
Community members contribute Apache licensed source code, ideas and functional direction via working groups etc.
Members such as IBM can then take the Fabric (and other Hyperledger projects) source code and use it
in delivering their own specific commercial blockchain-based infrastructure services
and/or work with clients to develop specific applications using Fabric as a foundation for distributed ledgers.

IBP - Chaincode development

체인코드는 여러 언어로 작성할 수 있으며 IBM Blockchain Platform에서는 Go와 Node.js로 작성된 체인코드를 지원합니다.
체인코드를 사용하면 Fabric 체인코드 인터페이스에서 제공하는 API를 사용하여 blockchain에 저장된 데이터를 조회하고 변경할 수 있습니다.
블록체인의 데이터는 채널 원장의 World State로 키-값 쌍에 저장됩니다. 체인코드는 get 명령을 사용하여 값을 검색하고 put 명령을 사용하여 값을 작성하거나 업데이트합니다.
이러한 기본 오퍼레이션을 사용하면 네트워크의 비즈니스 규칙을 정의하는 함수를 빌드할 수 있습니다.
이러한 함수는 애플리케이션에 의해 호출되어 네트워크의 일반 사용자에게 제공될 수 있습니다.
